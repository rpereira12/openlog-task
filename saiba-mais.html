Informações relevantes sobre os conceitos utilizados para desenvolver a página.

HTML5


Utilizado como linguagem de marcação.

CSS3


Utilizado para estilizar alguns elementos e fazer pequenas animações e transições.

Javascript


Utilizado junto ao jQuery para criar toda a logica da página de Login e página de cadastro de produtos. Detalhe para abordagem de Expressões Regulares (RegExp) utilizada para implementar a lógica de pesquisa na página de produtos.

Bootstrap


Utilizado o framework Bootstrap 3 para garantir otimização de tempo e recursos com a construção do layout e aumentar a qualidade. Também foi utilizado o Light Bootstrap Dashboard v1.4.0 do qual contém recursos padronizados para criação de um painel administrativo responsivo.
